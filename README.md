Rentopa — Simple House Hunting Demo

This folder contains a small static demo site for house hunting. Open `home.html` in a browser to view the site.

Files added:
- `home.html`, `about.html`, `contact.html`, `help.html`, `house.html`, `login.html`, `signup.html`
- `css/styles.css` — responsive styles using linear gradients
- `js/app.js` — client-side data and interactivity (render listings, house details, auth demo)
- `assets/house1.svg` ... `house4.svg` — simple SVG placeholders for homes

How to run:
- Open `home.html` in your browser (no server required).

Notes:
- This is a static demo: login/signup store demo users in `localStorage` only.
- `house.html?id=#` shows house details for id 1-4.
